## 🚀 Projects

### 📄 Resume Screening
> Automatically classify and screen resumes using NLP techniques.

- **Task:** Multi-class classification of resumes by job category
- **Techniques:** Text cleaning, TF-IDF Vectorization, Classification
- **Libraries:** NLTK, Scikit-learn
- **Notebook:** `Resume_Screening.ipynb`

---

### 🔑 Keyword Extraction
> Extract the most relevant keywords from raw text documents.

- **Task:** Identify and rank important terms from unstructured text
- **Techniques:** TF-IDF, Frequency Analysis
- **Libraries:** NLTK, Scikit-learn
- **Notebook:** `Keyword_Extraction.ipynb`

---

### 💬 Sentiment Analysis
> Classify text as Positive, Negative, or Neutral.

- **Task:** Sentiment classification on text data
- **Techniques:** Text preprocessing, Vectorization, Classification
- **Libraries:** NLTK, Scikit-learn, TensorFlow/Keras
- **Notebook:** `Sentiment_Analysis.ipynb`

---

### ❓ Quora Duplicate Question Detection
> Detect whether two questions asked on Quora are semantically identical.

- **Task:** Binary classification — duplicate or not duplicate
- **Techniques:** Text preprocessing, Feature Engineering, Deep Learning
- **Libraries:** NLTK, TensorFlow/Keras
- **Notebook:** `Quora_Duplicate_Questions.ipynb`
